# pulse-measure
- パソコンのカメラから映像を1フレーム毎に読み込み、その画像の平均の輝度値(R)を計測してグラフに出力するプログラム．
- jupyter notebook での使用を前提としている．

- 使用したライブラリ
  - openCV
  - mutplotlib
  - numpy

-使い方
  - 実行前にカメラに指を当てておく
  - 実行するとしばらくカメラが稼働する
  - 200個データをとるとカメラが止まり、グラフが作られる
  - カメラの画像が灰色の画面しか映さないときがあるが、実際は数値は取れているので特に問題ない．
